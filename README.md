# Testes em Java

Este repositório foi criado exclusivamente para fins de **estudo e aprendizado** da linguagem Java. O objetivo principal é praticar a lógica de programação,
orientação a objetos e o desenvolvimento de interfaces gráficas.

## Projeto Calculadora

O primeiro projeto desenvolvido e hospedado aqui é uma **Calculadora**. Trata-se de uma aplicação simples desenvolvida utilizando a IDE NetBeans para consolidar
os conceitos iniciais da linguagem.

### Funcionalidades Estudadas:
* Operações matemáticas básicas (Soma, Subtração, Multiplicação e Divisão).
* Construção de interface gráfica (telas, botões e campos de texto).
* Manipulação e tratamento de eventos de clique.
